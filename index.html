    Web 1.0 Web 2.0 Web 3.0
    ה-web מאפשר ביצוע פעולות קריאה בלבד. ה-web מאפשר ביצוע פעולות של קריה וכתיבה. ה-web מאפשר ביצוע פעולות של קריאה וכתיבה המותאמות אישית עבור המשתמש.
    Web 1.0 עוסק בתוכן סטטי אשר הפרסום בו הוא חד-כיווני של תוכן ללא כל אינטרקציה אמיתית בין קוראים, מפרסמים או ביניהם. Web 2.0 עוסק בתקשורת דו-כיוונית באמצעות רשתות חברתיות, בלוגים, תיוגים או תוכן ווידאו שנוצר על ידי משתמשים. Web 3.0 עדיין לא מוגדר באופן חד משמעי. בינה מלאכותית והאינטרנט לומדים את רצון המשתמש ומספקים לו חווית אינטרנט מותאמת אישית.
    מנועי החיפוש מאחזרים תוכן מאקרו. החיפוש מהיא מאוד אבל במרבית הפעמים תוצאותיו אינן מדוייקות. מנועי החיפוש מאחזרים תגיות עם תוכן מיקרו. תהליך התיוג נעשה באופן ידני, מייגע ומכסה אחוזים זניחים מה-WWW. מנועי החיפוש מאחזרין טקסטים של מיקרו - תוכן שתוייג באופן אוטומטי. הדבר מרמז על תרגום מיליארדי תכני מאקרו של web 1.0 לתוכן מיקרו.

    תרחיש שימוש: חיפוש השאילתה "איטליה"

    Web 1.0 - בכדי לבצע חיפוש עבור הערך "איטליה", המשתמש צריך להקליד כתובת URL מדויקת ולא יוכל להשתמש במנוע חיפוש כלשהו.
    Web 2.0 - בכדי לבצע חיפוש עבור הערך "איטליה", המשתמש יכול להשתמש במנוע חיפוש, הוא יקבל מספר תוצאות אך לא כולן יהיו בהכרח רלוונטיות.
    Web 3.0 - כדי לבצע חיפוש עבור הערך "איטליה", המשתמש יכול להשתמש במנוע חיפוש, הוא יקבל מספר תוצאות שונו עבור ערך זה. רוב התוצאות אשר יקבל יהיו רלוונטיות למשתמש הרגיל, וחלקן יהיו יותר מותאמות עבור המשתמש הספציפי שביצע את השאילתה.
